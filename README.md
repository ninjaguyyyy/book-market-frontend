## About Project

Project is produced for "Thiết kế phần mềm" project.<br/>
Deploy: https://bookstore-online.netlify.app/


## Features


`Buyer`
- See books (list, detail) are selling.
- See seller's information.

`Seller`
- Post a book to sell

`Admin`
- Manage buyers (get)
- Manage sellers (get, prohibit seller role)
